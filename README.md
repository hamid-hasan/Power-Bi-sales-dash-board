# Sales Analysis Dashboard (Power BI)

This is my first Power BI project after taking long break. I built an interactive sales dashboard using a supermarket sales dataset.

## Dashboard Features
- Total Sales KPI
- Total Orders KPI
- Total Customers KPI
- Total Products KPI
- Sales by Category
- Sales by Region
- Sales Trend by Year
- Sales by Sub-Category
- Interactive Filters

## Key Insights
- Phones were the top-selling sub-category.
- West region generated the highest sales.
- Sales grew significantly after 2016.
